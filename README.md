# ♟️ Sistema de Xadrez em Java

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos, simulando uma partida completa de xadrez no terminal.

## 📖 Sobre o Projeto

Este projeto implementa as regras oficiais do xadrez, incluindo movimentação das peças, controle de turnos, verificação de xeque e xeque-mate, e jogadas especiais como roque.

O objetivo principal foi praticar conceitos avançados de POO e estruturação de código em Java.

---

## 🚀 Tecnologias Utilizadas

* Java 17+
* Programação Orientada a Objetos
* Git e GitHub

---

## 🎮 Funcionalidades

* Movimentação completa de todas as peças:

  * Rei
  * Rainha
  * Torre
  * Bispo
  * Cavalo
  * Peão
* Validação de movimentos inválidos
* Sistema de turnos
* Detecção de xeque
* Detecção de xeque-mate
* Roque (pequeno e grande)

---

## 🧠 Conceitos Aplicados

* Encapsulamento
* Herança

* Sobrescrita de métodos

## ▶️ Como Executar

1. Clone o repositório:

```
git clone https://github.com/seuusuario/seurepositorio.git
```

2. Entre na pasta do projeto:

```
cd nome-do-projeto
```

3. Compile:

```
javac application/Program.java
```

4. Execute:

```
java application.Program
```


* `chess` → regras específicas do xadrez
